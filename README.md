# Claude Skills

A collection of Claude Code skills for specialized domains.

## Skills

### [Home Assistant Expert (Lite)](./home-assistant-expert-lite/)
Free. Gives Claude working knowledge of Home Assistant automation YAML — triggers, conditions, actions, and entity ID conventions. Write automations from plain English.

### [Home Assistant Expert](./home-assistant-expert/)
Full version. Adds Jinja2 templating, Zigbee2MQTT / Z-Wave / Ecobee / ESPHome / HACS integration deep-dives, Lovelace dashboard card patterns, and a step-by-step debugging protocol.

## Usage

Install via Claude Code skills:

```bash
claude skills install https://github.com/ktxkrew/claude-skills/tree/main/home-assistant-expert-lite
```
